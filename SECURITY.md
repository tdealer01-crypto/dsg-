# Security Policy

## Supported Versions

We currently support the following versions of DSG with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

We take the security of DSG seriously. If you believe you have found a security vulnerability, please report it to us by emailing security@dsg.pics.

**Please do not report security vulnerabilities through public GitHub issues.**

Once you have submitted your report, we will acknowledge receipt of your report within 48 hours and provide an estimated timeframe for a fix.
